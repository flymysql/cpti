# CPTI

CPTI 是一个移动端优先的恋爱人格测试静态产品原型，当前版本已拆分为独立的首页、测试页、结果页。

## 当前能力

- 独立页面结构：`首页` / `测试页` / `结果页`
- 测试页采用瀑布流短题卡
- 34 道恋爱向题目（含 4 道聊天场景题、2 道恶搞题）

- 多种答题形式：
  - 三选题
  - 四选题
  - 聊天场景题
- 双结果输出：
  - 你是哪种恋爱人格
  - 同一套人格里哪一型最适合接住你
- 20 个共享恋爱人格类型

- MBTI 可选输入，并作为轻度辅助权重参与计算
- 结果页支持结果文案复制

## 文件结构

- `index.html`：首页
- `quiz.html`：测试页
- `result.html`：结果页
- `styles.css`：共享视觉样式
- `data.js`：题库、人格类型和计算逻辑
- `quiz.js`：测试页交互
- `result.js`：结果页渲染
- `assets/`：历史占位资源

## 本地预览

直接用浏览器打开 `index.html` 即可，首页可跳转到测试页和结果页。
