## 使用说明

1. **单套人格**（渣男、渣女、daddy、男妈妈、妈妈）：任选一个代码块整段复制即可。  
2. **双套人格**：按生成目标选择 **男性向** 或 **女性向** 下的代码块整段复制；两套共享同一 `accent` hex，仅角色外形与英文关键词区分性别。  
3. 若 API 对中文支持弱，可只保留 **Optional EN** 段，或中英整段一起投喂（视模型说明而定）。  
4. `accent` 与仓库 `profiles/index.js` 中各人格 hex 对齐，可按需微调饱和度。
