## 使用说明

1. 复制某个代码块内的**整段文字**（含 Optional EN 或删除该段，仅保留中文亦可）。
2. 若 API 对中文支持弱，可**只保留 Optional EN 段**并把中文全局句换成上一版给你的英文全局句，或中英整段一起投喂（视模型说明而定）。
3. `accent` 已与仓库 `profiles/index.js` 中 hex 对齐，可按需微调饱和度。
